AutomateOps Toolkit - Embedding Options

Option 1: iframe Embedding

Use this option if you want to embed the UTM builder using an iframe:

<iframe 
    src="utm_builder.html" 
    style="width: 100%; height: 800px; border: none;"
    title="AutomateOps UTM Builder"
></iframe>
Option 2: Direct HTML Embedding

Use this option if you want to embed the UTM builder directly in your HTML:

Download UTM Builder HTML

Open the downloaded file and copy its contents into your Softr HTML element.

Additional Notes

  • The iframe option is easier to maintain as it loads the content dynamically.
  • Direct HTML embedding gives you more control over the styling and behavior.
  • Both options include all necessary CSS and JavaScript - no additional dependencies required.
